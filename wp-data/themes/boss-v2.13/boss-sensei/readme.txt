Boss for Sensei integrates the Sensei LMS with the Boss theme.

== Description ==

Make Sensei beautiful on BuddyPress with our Boss theme integration.

Just activate the plugin, and all of your Sensei content will match the Boss theme.

== Installation ==

1. Make sure BuddyPress, Sensei, and Boss theme are activated.
2. Visit 'Plugins > Add New'
3. Click 'Upload Plugin'
4. Upload the file 'boss-sensei.zip'
5. Activate Boss for Sensei from your Plugins page.
